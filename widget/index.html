Foyer widget — local dev page

The widget mounts itself on this page. Run npm run build first to produce dist/foyer-widget.js, then open this file via a static server (e.g. python3 -m http.server in the widget directory).

Override the API base via data-foyer-api-base below — by default it points at the production host, which does not exist yet for local dev. Point it at http://localhost:8000 when the Laravel php artisan serve is running.
